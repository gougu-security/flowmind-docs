# 许可与激活

FlowMind 部分功能需有效许可后使用。许可相关操作均在应用内 **设置 → 许可** 完成，无需手动编辑系统文件。

::: warning 安全提示
- 许可文件（`.license`）与申请文件（`.license-request`）仅供您本人或授权管理员使用，**请勿公开分享或上传到公共仓库**。
- 若许可已绑定设备，请勿尝试在非授权机器上使用同一文件。
- 遇到问题时联系产品管理员或技术支持，**不要**在公开渠道粘贴完整许可内容或设备标识。
:::

## 查看当前状态

在 **设置 → 许可** 顶部卡片可查看：

| 状态 | 含义 |
|------|------|
| 未激活 | 尚未导入有效许可 |
| 有效 | 许可可用，按授权范围开放功能 |
| 已过期 | 许可已超过有效期 |
| 设备不匹配 | 许可与当前设备绑定关系不符 |
| 已吊销 | 许可已被撤销 |

卡片中还会显示版本计划（如试用版、专业版等）、剩余天数及当前授权的功能列表（以应用内展示为准）。

点击 **刷新状态** 可重新读取本地许可信息。

## 导入许可

若已从管理员处获得许可文件：

1. 打开 **设置 → 许可**
2. 在 **导入许可文件** 区域点击 **选择并导入许可文件**
3. 选择扩展名为 `.license` 的文件
4. 根据提示确认导入结果

导入成功后，相关功能将按授权自动启用；若提示无效或设备不匹配，请联系签发方重新申请。

## 申请新许可

尚未持有许可文件时，可在应用内生成申请：

1. 在 **生成许可申请** 区域填写 **客户名称** 与 **客户邮箱**
2. 可选填写备注（如试用、内部测试）
3. 点击 **生成许可申请**
4. 将生成的 `.license-request` 文件发送给管理员或销售/support 渠道

::: tip
申请文件包含设备相关信息，仅用于签发流程。发送时使用您与管理员约定的**私密渠道**（邮件、工单等），避免在公开论坛或聊天群传播。
:::

收到签发的 `.license` 文件后，按上一节 **导入许可** 操作即可。

## 移除许可

如需在本机停用当前许可（例如更换账号或移交设备）：

1. 在状态卡片中点击 **移除许可**
2. 确认后功能将恢复为未激活/受限状态

移除操作仅影响本机，不会自动向服务器注销；重新激活需再次导入有效许可文件。

## 常见问题

### 导入后仍显示未激活或无效

- 确认文件完整、未被改名或损坏
- 点击 **刷新状态** 后重试
- 仍失败请联系签发方，**提供申请文件或工单编号**，勿在公开场合发送许可全文

### 提示设备不匹配

说明该许可与当前设备绑定策略不一致。请联系管理员重新签发或确认是否使用了通用许可。

### 功能入口灰色或提示升级

当前许可未包含该功能。可在许可卡片中查看已授权功能列表，或联系管理员调整授权范围。

### 许可过期

联系管理员续期或重新签发；过期后部分高级功能可能不可用，基础能力以应用内提示为准。

## 获取帮助

- 企业/商业用户：通过采购或技术支持渠道联系管理员
- 个人试用：在应用内生成许可申请并按要求提交
