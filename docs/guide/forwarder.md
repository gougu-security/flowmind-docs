# 转发器

转发器是 FlowMind 的核心界面，用于实时查看和分析被代理的 HTTP/HTTPS/WebSocket 流量。

## 界面布局


![FlowMind 转发器界面](/screenshots/forwarder-home.png)


## 流量列表

### 列信息

| 列名 | 说明 |
|------|------|
| 方法 | HTTP 方法（GET、POST、PUT 等） |
| URL | 请求的完整 URL |
| 状态码 | HTTP 响应状态码 |
| 大小 | 响应体大小 |
| 耗时 | 请求耗时（毫秒） |
| 时间 | 请求时间戳 |

### 排序与选择

- **排序**：点击列头进行升序/降序排序
- **选择**：点击行选中，支持多选（Ctrl/⌘ + 点击）
- **全选**：Ctrl/⌘ + A

## 过滤功能

### 方法过滤

选择特定 HTTP 方法进行过滤：

- GET、POST、PUT、DELETE、PATCH、HEAD、OPTIONS

### 状态码分组

按状态码范围过滤：

| 分组 | 范围 |
|------|------|
| 信息 | 1xx |
| 成功 | 2xx |
| 重定向 | 3xx |
| 客户端错误 | 4xx |
| 服务端错误 | 5xx |

### 搜索过滤

在搜索框输入关键词，会匹配以下字段：

- URL
- 请求头/响应头名称和值
- 请求体/响应体内容

### 域名过滤

按请求域名进行过滤，支持：

- 精确匹配：`api.example.com`
- 通配符匹配：`*.example.com`

## 详情面板

选中流量后，详情面板显示以下标签页：

### 概览

显示请求的基本信息：

- 请求方法和 URL
- 状态码和响应大小
- 请求/响应时间
- 客户端和服务器地址

### 请求

- **请求头**：完整的请求头列表
- **请求体**：请求体内容（支持 JSON、表单、原始格式）

### 响应

- **响应头**：完整的响应头列表
- **响应体**：响应体内容（支持 JSON、HTML、原始格式）

### WebSocket

仅对 WebSocket 流量显示：

- **消息列表**：所有 WebSocket 消息
- **消息详情**：选中消息的内容预览

### TLS

显示 TLS/SSL 连接信息：

- 协议版本
- 密码套件
- 证书信息
- SNI（Server Name Indication）

## 右键菜单

在流量列表中右键点击，可以：

| 操作 | 说明 |
|------|------|
| 发送到重发器 | 将请求发送到重发器进行重放 |
| 发送到模糊器 | 将请求发送到模糊器进行测试 |
| 发送到 AI | 将请求发送给 AI 进行分析 |
| 复制 URL | 复制请求 URL |
| 复制 cURL | 复制为 cURL 命令 |
| 从列表移除 | 从当前列表中移除（不删除数据） |

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl/⌘ + F` | 聚焦搜索框 |
| `Ctrl/⌘ + K` | 清空过滤条件 |
| `Delete` | 删除选中流量 |
| `Ctrl/⌘ + A` | 全选 |
| `↑` `↓` | 切换选中项 |
