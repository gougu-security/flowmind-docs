# 设置

FlowMind 的设置页采用**左侧导航 + 右侧内容区**布局。修改配置后，需点击右上角 **保存设置** 才会写入本地配置。

## 设置导航

| 标签 | 说明 |
|------|------|
| **通用** | 主题、日志级别、请求体/WebSocket 捕获上限、本地工作区路径 |
| **代理** | 监听端口、监听地址、上游代理 |
| **浏览器** | 默认浏览器、自定义路径、无痕模式、CDP 调试端口 |
| **证书** | CA 路径、导出/重新生成、国密 SPKI 显示 |
| **许可** | 导入许可、申请激活与状态查看（见 [许可与激活](./license.md)） |
| **AI** | 对话模型与扩展能力（见下文） |
| **关于** | 应用版本信息 |

::: tip 工作区路径
在 **通用 → 本地工作区** 可查看并打开应用数据根目录、配置文件、数据库、证书目录、浏览器配置根目录、**当前项目**浏览器配置目录、报告与插件工作区等路径。
:::

## 通用

| 配置项 | 说明 |
|--------|------|
| 主题 | 暗色 / 亮色 / 跟随系统 / 自定义（含预设与颜色微调） |
| 日志级别 | Info 或 Debug |
| 请求体最大捕获大小 | 单位 MB，默认 10 |
| WebSocket 消息最大捕获大小 | 单位 MB，默认 1 |

## 代理

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| 代理端口 | `8080` | 本地 MITM 监听端口 |
| 监听地址 | `127.0.0.1` | 绑定 IP |
| 上游代理 | 空 | 可选，如 `http://proxy.corp:8080` |

::: info 域名过滤
按域名排除抓包在 **转发器** 的过滤栏中配置，不在设置页。
:::

## 浏览器

| 配置项 | 说明 |
|--------|------|
| 默认浏览器 | 自动检测本机 Chrome / Firefox / Edge 等 |
| 自定义浏览器路径 | 覆盖自动检测的可执行文件路径 |
| 无痕/隐私模式 | 启动时附加隐私参数 |
| 下次启动前自动清理当前项目的浏览器配置目录 | 启动浏览器前删除当前项目的 Profile（`browser-profile/{项目ID}/`），不影响其他项目 |
| 额外启动参数 | 如 `--window-size=1280,720` |
| CDP 远程调试端口 | Chromium 系 Browser Agent 接入点，常用 `9222` |

点击标题栏 **启动浏览器** 时，FlowMind 会自动注入代理与证书相关参数。

## 证书

| 操作 | 说明 |
|------|------|
| 查看 CA 路径 | 首次启动代理后自动生成 |
| 导出根证书 | 导出到证书目录，供系统/浏览器安装 |
| 重新生成 CA | 替换密钥对；旧证书需重新安装 |
| CA 密钥密码 | 可选，保存在本机配置 |
| 国密 SPKI 指纹 | 仅影响指纹显示算法（SM3），不改变签名算法 |

详细安装步骤见设置页 **证书** 标签内的平台指引。

## AI 配置

AI 设置分为 **对话模型**（主页面内联）与 **扩展配置**（卡片 + 弹窗）两部分。

### 对话模型

| 配置项 | 说明 |
|--------|------|
| 活跃 Provider | 当前对话使用的模型提供商 |
| 模型 | 从 Provider 列表选择或手动输入模型名 |
| Max Tokens | 单次回复 token 上限 |
| Temperature | 采样温度 0–2 |
| 工具调度上限 | 每轮用户消息内，模型连续发起工具调用的最大次数 |
| 上下文窗口解析 | 点击「重新解析」预览当前模型上下文上限 |

### 扩展配置（弹窗）

点击对应卡片打开弹窗编辑；弹窗内修改即时写入当前设置草稿，关闭后仍需 **保存设置**。

| 卡片 | 内容 |
|------|------|
| **Provider** | API Key、Base URL、可用模型列表、Provider 级默认上下文 |
| **Embedding** | 向量模型、维度、Provider、测试连接与启用开关 |
| **MCP Servers** | Stdio / HTTP-SSE 传输、重连状态 |
| **动态工具** | Shell 子进程工具、参数 Schema |
| **AI 角色** | 内置/自定义人设、默认活跃角色、系统提示词覆盖 |
| **自定义 Skill** | 对话页 Skill 扩展、工具范围与自动上下文 |
| **高级选项** | 未知模型假定上下文、Ollama `/api/show`、Python 沙箱路径 |

#### Embedding 快速配置

1. 打开 **Embedding** 卡片
2. 选择 Provider，填写模型名与向量维度（可点「填入推荐」）
3. 点击 **测试连接**
4. 勾选 **启用 Embedding**
5. 保存设置

知识库、向量记忆与对话 RAG 均依赖 Embedding 就绪。

#### MCP 示例（Stdio）

在 **MCP Servers** 弹窗中添加：

| 字段 | 示例 |
|------|------|
| 名称 | `filesystem` |
| 传输 | Stdio |
| 命令 | `npx` |
| 参数 | `-y, @modelcontextprotocol/server-filesystem, /path/to/dir` |

保存后可在弹窗内 **重连**，查看已连接 Server 与工具数量。

## 配置文件

部分高级项保存在应用数据目录下的配置文件中，由应用在保存设置时自动维护。**不建议**手动编辑，以免与界面配置冲突。若需批量部署或企业定制，请联系技术支持。

## 故障排除

### 配置不生效

1. 确认已点击 **保存设置**
2. 重启应用并查看日志（通用 → 日志级别设为 Debug）

### 证书 / HTTPS 抓包失败

1. 确认 CA 已安装到**受信任根证书**存储
2. 重新导出并安装（若曾重新生成 CA）
3. Firefox 需单独导入证书库

### AI 无法连接

1. 在 Provider 弹窗检查 API Key 与 Base URL
2. Embedding 需单独测试并通过后再启用
3. MCP 需在弹窗内重连并查看连接状态
