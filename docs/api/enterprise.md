# 企业版本

FlowMind 企业版为大型组织提供全方位的应用安全解决方案。

## 核心优势

### 团队协作

- **多用户管理**：支持角色权限控制（管理员、分析师、审计员）
- **项目隔离**：不同团队的项目数据完全隔离
- **协作工作流**：支持发现分配、状态流转、备注协同

### 集中式管理

- **统一控制台**：一站式管理所有扫描任务和结果
- **策略下发**：总部统一配置扫描策略，分支机构执行
- **资产管理**：自动发现和管理企业数字资产

### 安全合规

- **审计日志**：所有操作可追溯，满足等保、ISO27001 要求
- **数据主权**：支持私有部署，数据不出企业边界
- **SSO 集成**：支持 LDAP、OAuth2、SAML 等企业认证体系

## 技术规格

| 指标 | 社区版 | 企业版 |
|------|--------|--------|
| 并发扫描数 | 1 | 无限制 |
| 历史数据保留 | 本地 SQLite | 企业级数据库 |
| 用户数 | 单用户 | 无限制 |
| API 调用限制 | 有 | 无限制 |
| 扫描规则数 | 社区规则库 | 社区 + 企业专属规则 |
| AI 模型 | 公共模型 | 可私有化部署 |

## 部署方式

### SaaS 云服务

- 即开即用，无需运维
- 数据加密存储，通过安全审计
- 按需付费，弹性扩展

### 私有化部署

- 部署在企业自有服务器或私有云
- 完全掌控数据和配置
- 支持离线环境部署

### 混合部署

- 核心数据本地存储，AI 能力云端调用
- 兼顾安全性和功能性

## 行业解决方案

### 金融行业

- 满足 PCI DSS、等保三级要求
- 针对金融业务的专属扫描规则
- 交易链路安全分析

### 政企行业

- 满足等保 2.0 要求
- 国密算法支持（SM2/SM3/SM4）
- 信创环境适配

### 互联网行业

- DevSecOps 流水线集成
- API 安全全生命周期管理
- 业务逻辑漏洞检测
